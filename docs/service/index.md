# 服务帮助说明
为帮助用户理解 ATRI 各个服务食用方法，特此写明。

## 文档服务帮助格式
```
# （服务名）
（服务文档介绍）

## 服务信息
- **服务名**：（服务名称）
- **说明**：（服务说明）
- **权限限定**：（注明大体最低使用权限）

## 命令列表
（服务内各项命令列表）
- **（命令）、（同功能的其它命令）**（附加说明）：（命令说明）
......
```

## 补充
文档只列举了可通过命令触发的对应服务帮助，以最大化阅读效率。

被动触发的功能，例如 [kimo](./kimo.md) 中的文爱，触发方式为被动检测，详细请通过服务 [帮助](./help.md) 查阅。
