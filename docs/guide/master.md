# 主人食用指南

## 说明
该栏目所提到的服务，为主人**可使用、可触发、相关的**的。

如果**输入对应命令、提供可能的触发信息**后没有反应，可能是当前所在的群聊**禁用**，或者 ATRI 的主人**全局禁用**、又或者如指南首页提到的**代码层面禁用**了该服务。
